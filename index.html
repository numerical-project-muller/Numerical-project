Exploring Muller's Method

An advanced analysis, proof, and engineering application of parabolic interpolation for solving real and complex mathematical roots.

1. Background

What is a Root-Finding Problem?

In numerical mathematics, a root-finding problem involves finding values of x = c (roots) that satisfy the equation f(c) = 0 for a continuous function f(x). Geometrically, finding a root corresponds to identifying the exact point where the curve of the function intersects or touches the x-axis.

Why It Matters

Most real-world physical systems are governed by non-linear relationships. Since analytical algebraic formulas only exist for equations of degree 4 or lower, engineers rely heavily on robust numerical approximation methods.

Milestones in Root-Finding History

Ancient Babylonians Square Root Approximations First recorded steps in solving quadratic relations.
17th Century Newton-Raphson Method Introduced linear derivatives to achieve quadratic convergence for real roots.
1956 (Modern Era) David E. Muller's Method Introduced quadratic interpolation (parabolic fitting) to naturally resolve both real and complex conjugate roots.

Approximating roots step-by-step using algorithmic loops. While methods like Bisection and Newton's stay on the real axis, Muller's Method naturally handles complex numbers without needing complex initial guesses.

3. Step-by-Step Algorithm

Muller's method uses three initial approximations x0, x1, and x2 to construct a unique moving quadratic polynomial P(x) = a(x - x2)2 + b(x - x2) + c that passes through (x0, f(x0)), (x1, f(x1)), and (x2, f(x2)). The next approximation x3 is estimated by finding where this parabola crosses the axis.

1

Difference Formulas

Calculate the step intervals and slope rates:
h0 = x1 - x0,      h1 = x2 - x1
δ0 = f(x1) - f(x0) h0 ,      δ1 = f(x2) - f(x1) h1
2

Parabolic Coefficients

Compute coefficients a, b, and c for the interpolating parabola:
a = δ1 - δ0 h1 + h0
b = ah1 + δ1,      c = f(x2)
3

Root Correction

Calculate next point. Choose the sign in the denominator to maximize its absolute value:
x3 = x2 - 2c b ± √b2 - 4ac

4. Mathematical Examples

Example 1: Comparing Newton's Method and Muller's Method

1. Problem Statement

Solve the following cubic polynomial equation to find its real root:
f(x) = x3 - 13x - 12 = 0
Analytical (Exact) Solution: By factoring the polynomial, we obtain (x - 4)(x + 1)(x + 3) = 0. Thus, the exact roots are x1 = -3, x2 = -1, and x3 = 4. In this example, we will trace and compare how both numerical methods converge toward the target exact root, x = 4.

2. Solution by using Newton's Method

Let x0 = 5.0, and the derivative is f '(x) = 3x2 - 13. The Newton's Recurrence Formula is:
xn+1 = xn - f(xn) f '(xn) = xn - xn3 - 13xn - 12 3xn2 - 13

Iteration 1: Substitute x0 = 5.0 into the function and its derivative:

f(5.0) = (5.0)3 - 13(5.0) - 12 = 48

f '(5.0) = 3(5.0)2 - 13 = 62

Apply the formula to find x1: x1 = 5.0 - 48 / 62 = 4.225806

Iteration 2: Substitute x1 = 4.225806 into the formula:

f(4.225806) = (4.225806)3 - 13(4.225806) - 12 = 8.526602

f '(4.225806) = 3(4.225806)2 - 13 = 40.572321

Calculate x2: x2 = 4.225806 - 8.526602 / 40.572321 = 4.015648

3. Solution Using Muller's Method

Let x0 = 3.0, x1 = 3.5, x2 = 4.5. Corresponding function values are:
f(x0) = (3.0)3 - 13(3.0) - 12 = -24
f(x1) = (3.5)3 - 13(3.5) - 12 = -14.625
f(x2) = (4.5)3 - 13(4.5) - 12 = 20.625

Iteration 1 Step-by-Step:

i. Calculate differences (h0, h1) and slopes (δ0, δ1):

h0 = x1 - x0 = 0.5

h1 = x2 - x1 = 1.0

δ0 = (f(x1) - f(x0)) / h0 = (-14.625 + 24) / 0.5 = 18.75

δ1 = (f(x2) - f(x1)) / h1 = (20.625 + 14.625) / 1.0 = 35.25

ii. Compute coefficients (a, b, c) for the parabola:

a = (δ1 - δ0) / (h1 + h0) = (35.25 - 18.75) / (1.0 + 0.5) = 11.00

b = ah1 + δ1 = 11.00(1.0) + 35.25 = 46.25

c = f(x2) = 20.625

iii. Evaluate the discriminant (D):

D = b2 - 4ac = 46.252 - 4(11.0)(20.625) = 1231.5625 ⇒ √D = √1231.5625 = 35.0936

iv. Maximize the denominator:

Denominator = b + √D = 46.25 + 35.093625 = 81.343625

v. Calculate the next approximation (x3):

x3 = x2 - 2c / Denominator = 4.5 - 2(20.625) / 81.343625 = 3.992892

4. Discussions and Interpretations for Example 1

4.1 Convergence Rate and Error-Decay Efficiency: Based on the empirical numerical data consolidated in Table 2 and Table 3, both Newton's method and Muller's method successfully converged to the exact analytical root x = 4.000000. However, their internal error-decay dynamics exhibit a striking contrast in terms of speed and efficiency.

Newton's method commenced with a substantial true relative error of 25.000% at the initial guess x0 = 5.0. Iterating through linear tangent updates, the true relative error dropped to 5.645% in Iteration 1, and required 4 full iterations to completely eliminate the approximation error down to 0.000%.

In contrast, Muller's method displayed a significantly more aggressive convergence trajectory. Starting from three moving initial approximations (x0 = 3.0, x1 = 3.5, x2 = 4.5), its very first iteration successfully slashed the true relative error down to an astonishing 0.177% (x3 = 3.992892). By Iteration 3, Muller's method achieved perfect convergence to 4.000000 with a true relative error of 0.000%, effectively outpacing Newton's method by one full iteration.

4.2 Mathematical Interpretation of Algorithm Mechanisms: The structural reason behind Muller's superior error-clearing capacity lies in the geometric interpolation alignment of the two algorithms:

  • Newton's Method (Linear Tangent Approximation): This open method relies on a first-order Taylor series expansion, constructing a straight tangent line from a single localized point to project the next root. While it guarantees quadratic convergence (r = 2.0) when initialized sufficiently close to the root, its efficiency lags if the initial guess behaves poorly or carries an inflection drift relative to the function's global behavior.
  • Muller's Method (Quadratic Parabolic Approximation): This method utilizes three sequential data points to construct a unique moving quadratic polynomial:
    P(x) = a(x - x2)2 + b(x - x2) + c
    Because our target function f(x) = x3 - 13x - 12 is a cubic polynomial, a curved parabolic interpolation naturally accommodates and mirrors the local curvature of the function significantly tighter than a straight linear tangent line ever can.

This geometric advantage mathematically justifies why Muller's method yielded an absolute error of merely 0.007108 in its opening step. It empirically verifies that Muller's theoretical near-quadratic superlinear convergence rate (r ≈ 1.84) can match or exceed Newton's method when solving polynomial systems, all while operating completely derivative-free (eliminating the need to evaluate f '(x)).

Complex Root Core Feature

Example 3: f(x) = x2 + x + 1 = 0

Exact Roots: x = -0.5 ± 0.866025i. This exemplifies Muller's primary selling point over traditional methods.

Why Bracketing & Newton's Fail:

Bracketing methods require a sign change (f(a)f(b) < 0). However, since f(x) = (x + 0.5)2 + 0.75 > 0 for all real x, the parabola never intersects the real x-axis, making brackets impossible.

Similarly, Newton's and Secant methods initialized with real parameters are restricted to the real number line due to real-valued derivatives and slopes.

How Muller's Resolves This:

Muller's solver applies the quadratic root formula where the discriminant D = b2 - 4ac < 0 sits inside a square root. If D < 0, taking the square root automatically introduces the imaginary unit i, naturally steering the iterations into the complex plane even with purely real initial guesses.

Mathematical One-Step Convergence:

Setting initial real guesses: x0 = -1, x1 = 0, x2 = 1 ⇒ f(-1) = 1, f(0) = 1, f(1) = 3.
a = 1.0,      b = 3.0,      c = 3.0
D = b2 - 4ac = 9 - 12 = -3 ⇒ √D = √3i ≈ 1.732051i
Choosing the positive denominator: Denom = 3 + √3i.
x3 = 1 - 6 3 + √3i = 1 - 6(3 - √3i) (3 + √3i)(3 - √3i) = 1 - 18 - 6√3i 12 = -0.5 + 0.866025i

5. Real-World Application

Structural Vibration & Seismic Resonance Analysis

In structural engineering, analyzing the dynamic behavior of multi-story buildings subjected to seismic ground motions is a critical safety task. Consider a three-story shear frame building under lateral dynamic loading. To design against resonance, engineers must calculate the eigenvalues of the system to determine its natural frequencies. By formulating the characteristic determinant det(K - ω2M) = 0, where M is the lumped mass matrix and K is the structural stiffness matrix, and letting λ = ω2, we obtain the following cubic polynomial governing the structural frequency:
P(λ) = λ3 - 250λ2 + 15000λ - 200000 = 0
We need to find the fundamental eigenvalue (the smallest positive real root λ1) to prevent structural damage.

Iterative Process (Guesses: λ0 = 10, λ1 = 15, λ2 = 20):

Iteration 1: Intervals h0 = 5, h1 = 5. Slopes δ0 = 9225, δ1 = 7175. Polynomial values P(10) = -74000, P(15) = -27875, P(20) = 8000.
a = -205.000000,      b = 6150.000000,      c = 8000.000000
D = b2 - 4ac = 44382500 ⇒ √D ≈ 6661.996728
Choosing the positive denominator: Denom = 12811.996728 ⇒ λ3 ≈ 18.751167 (εa = 6.66%).
Iteration 2: Updated values: λ0 = 15, λ1 = 20, λ2 = 18.751167 ⇒ P(18.751167) ≈ -47.753800. Repeating the algorithm yields:
a ≈ -194.811000,      b ≈ 6687.521000,      c ≈ -47.753800
λ3 ≈ 18.758312      (εa = 0.038%)

Engineering Interpretations:

Using Muller's method, we resolved the lowest eigenvalue λ1 ≈ 18.7583 rad2/s2. This translates directly to the fundamental natural frequency of the structure:
ω1 = √λ1 = √18.7583 ≈ 4.331 rad/s
Converting this to cyclic frequency:
f1 = ω12π ≈ 0.689 Hz

Significance in Seismic Design: Most earthquake waves carry high energy within the 0.5 Hz to 5.0 Hz frequency range. Since our building's fundamental frequency f1 ≈ 0.689 Hz lies squarely in this band, there is an extreme risk of resonance causing failure. To secure the building, engineers would use this numerical feedback to either stiffen the columns to shift the frequency above 1.5 Hz, or incorporate base isolation rubber bearings to safely lower the system's frequency below 0.2 Hz.
